Optional emitOutsideAngularZone

emitOutsideAngularZone?: boolean

Controls if to emit received messages inside or outside the Angular zone.

By default, if not specified, emits in the zone in which subscribed to the Observable.

Deprecated

since version 17.1.0; Messages are received in the zone in which subscribed to the Observable. To receive messages outside the Angular zone, subscribe to the Observable outside the Angular zone, otherwise inside the Angular zone; API will be removed in a future release.

Optional subscribeTimeout

subscribeTimeout?: number

Specifies the maximum time (in milliseconds) to wait for the destination to be subscribed. If specified, overrides the global timeout as set via SessionProperties#readTimeoutInMsecs.
