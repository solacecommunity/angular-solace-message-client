Interface SDTUnsupportedValueError

Represents a SDT unsupported value error.

An SDT field was assigned a value that is within the type range for the given SDT type, but is not supported on this platform/runtime.

This occurs when a received {@link SDTContainerMap} or {@link SDTContainerStream} contains a field with a value that can not represented in the local architecture.

Possible causes include:

  • receive 64 bit integer that cannot be represented accurately in a javaScript number. JavaScript numbers are floats and can only hold a 48 bit integer without loss of precission. Any integer greater than 281474976710655 or less than -281474976710655 will cause this exception.
