Interface QueueDescriptor

Identifies a queue or topic endpoint on the message router.

Operations that make use of queue descriptors include

  • {@link Session#createMessageConsumer}
  • {@link Session#dteUnsubscribe}

For durable endpoints the name of that endpoint needs to be specified. For non-durable endpoints, the name is generated when the MessageConsumer is connected. The generated descriptor can be queried from the consumer after it has successfully connected by calling {@link MessageConsumer#getProperties#queueDescriptor}.
