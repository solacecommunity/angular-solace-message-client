acknowledgeMode

acknowledgeMode: MessagePublisherAcknowledgeMode
  • Defined in lib/solace.model.ts:362

The message-router sends windowed acknowledgements which the API converts to per-message acknowledgement by default. If acknowledgeMode is Windowed, then the API will simply pass through the message-router acknowledgements.

default

MessagePublisherAcknowledgeMode.PER_MESSAGE

acknowledgeTimeoutInMsecs

acknowledgeTimeoutInMsecs: number
  • Defined in lib/solace.model.ts:354

The time to wait for an acknowledgement, in milliseconds, before retransmitting unacknowledged messages. The valid range is 20 <= value <= 60000.

default

2000

enabled

enabled: boolean
  • Defined in lib/solace.model.ts:338

When enabled, a Guaranteed Messaging Publisher is automatically created when a session is connected.

The default value is the same as the value provided to {@link solace.SolclientFactory.init}, in the profile, {@link solace.SolclientFactoryProperties.profile}, in the field {@link solace.FactoryProfile#guaranteedMessagingEnabled}.

windowSize

windowSize: number
  • Defined in lib/solace.model.ts:346

Maximum number of messages that can be published without acknowledgment. The valid range is 1 <= value <= 255
