Enumeration QueuePermissions

Represents the permissions applicable to a queue.

The corresponding endpoint property is {@link QueueProperties#permissions}.

The access controls:

  • the permissions for all other users of the queue, this only applies to non-durable queues {@link QueueProperties#permissions};
  • for the current Message Consumer on a queue or endpoint, MessageConsumer.permissions

For example, creating a temporary topic endpoint with MODIFY_TOPIC will allow other users to modify the topic subscribed to that endpoint.
