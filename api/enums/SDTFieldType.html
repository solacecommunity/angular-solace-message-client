BYTEARRAY

BYTEARRAY: = 11
  • Defined in lib/solace.model.ts:1222

Maps to a Uint8Array.

Backward compatibility note: Using the version_10 factory profile or older, the getValue() method of a BYTEARRAY sdtField returns the byte array in 'latin1' String representation. Later profiles return a Uint8Array (in the form of a nodeJS Buffer instance in fact)

When creating a field of type BYTEARRAY, the following datatypes are all accepted as value:

  • Buffer (the nodeJS native type or equivalent)
  • ArrayBuffer,
  • Any DataView or TypedArray,
  • 'latin1' String for backwards compatibility: Each character has a code in the range 0-255 representing exactly one byte in the attachment.
