accessToken?: string | Type<OAuthAccessTokenProvider>

Specifies the access token required for OAUTH2 authentication. This is only relevant if the {@link AuthenticationScheme#OAUTH2} authentication scheme is being used.

We recommend specifying a OAuthAccessTokenProvider to continuously provide a valid access token.

See OAuthAccessTokenProvider for detailed instructions how to create and register an access provider.
